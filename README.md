# MaterialMusics
A material design music player for Android.


## Features
- .m3u playlists support
- ReplayGain support
- Group by Album, Artist, Folder
- cover.png Album/Folder support

## Building

Prerequisites :
⋅⋅⋅ - Java 1.8 JDK
⋅⋅⋅ - Android SDK

1) `gradle build` to build the APK
2) `gradle installDebug` to install the APK
